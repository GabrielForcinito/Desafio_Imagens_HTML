# 🖼 Desafio das Imagens

## 📌 Sobre o projeto
Projeto desenvolvido em HTML com o objetivo de praticar a inserção de imagens e favicon em páginas web.

## 🛠 Tecnologias utilizadas
- HTML5

## 🎯 Funcionalidades
- Inserção de imagem no conteúdo da página  
- Uso de favicon (ícone na aba do navegador)

1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` em seu navegador

## 📚 Aprendizado
Neste projeto pratiquei:
- Uso da tag `<img>`
- Caminho de arquivos (relativo)
- Inserção de favicon
